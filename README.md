## Requirement
- Angular : 17.3.5
- Angular CLI : 17.3.5
- Nodejs : 22.1.0
- npm : 10.7.0
- Module Federation Plugin

## Installation

1. Clone the repository:
   ```sh
    git clone https://github.com/your-username/project-name.git
    ``` 
2. Install the dependencies
    ```sh
    cd project_name
    npm install
    ```
## Usage
1. Start the development server:
    ```sh
    npm start
    ```
2. Open your browser:
   ```
   http://localhost:4200
   ```

3. Build the project for production:
    ```sh
    ng build
    ```
